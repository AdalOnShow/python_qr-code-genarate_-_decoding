# QR Code and Barcode Generator / Decoder

This project is a Python learning exercise focused on object-oriented
programming (OOP). It aims to build a simple application that can generate and
decode QR codes and barcode.

## Features

- Generate QR codes from text or URLs
- Generate barcode from numeric or text data
- Decode QR codes from images
- Decode barcode from images
- Built with Python and OOP principles

## Learnings

- Python classes and objects
- Encapsulation and modular design
- Working with image generation and decoding libraries
- Handling file input/output for barcode and QR code images

## Usage

1. Clone the repository.
2. Install dependencies (for example via `pip`).
3. Run the generator or decoder scripts.

## Notes

This repository is mainly for learning and experimentation with Python OOP and
barcode/QR functionality.
